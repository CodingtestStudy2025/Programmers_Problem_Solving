<접근법>
```
앞에서부터 살펴보면서 괄호 열리면 +1 닫히면 -1 해서 최종 0이 되도록
```

```python
def solution(s):

    # 0으로 초기화
    answer = 0

    for char in s:
        # 열리면 +1 닫히면 -1
        if char == '(':
            answer += 1
        elif char == ')':
            answer += -1

        # 닫힌게 더 많으면 False
        if answer < 0:
            return False

    # 제대로 안 닫혔으면 False
    if answer != 0:
        return False

    return True
```
