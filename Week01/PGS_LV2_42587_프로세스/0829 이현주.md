<접근법>
```
원형 큐
일단 앞에부터 탐색하면서 그 priority값이 현재 queue에 있는 값들 중 max에 해당하는 값이 아니라면 빼서 맨 뒤로 보냄 (priority queue와 매칭되는 process idx queue도 병행해서 관리.. 걍 이따구로 구현함)
그리고 순차적으로 아닌 애들 뒤로 빼다가 max인 애 만났다? 걔의 process idx를 order list에 append 해준다음 걔는 이제 다신 볼일 없으니까 priority값 0으로 바꿔줌
이 과정을 order_list에 모든 process idx가 들어올때까지 반복해줌

그리고 정답으로는 order_list에 location값이 몇번째 index에 있는지 반환해주면 됨 (제로 인덱싱이므로 +1 해서 반환해줌)
```

```python
def solution(priorities, location):
    # 원형 큐
    answer = 0
    
    process_order = []
    process_idx = []
    
    for i in range(0, len(priorities)):
        process_idx.append(i)
    
    while(len(process_order) != len(priorities)):
        if 0 != priorities.index(max(priorities)):
            tmp = priorities
            priorities = tmp[1:]
            priorities.append(tmp[0])
            
            tmp = process_idx
            process_idx = tmp[1:]
            process_idx.append(tmp[0])
        else:
            process_order.append(process_idx[0])
            priorities[0] = 0
            
    answer = process_order.index(location)+1
    
    return answer
```
